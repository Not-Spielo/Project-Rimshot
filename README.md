## Instructions

1. Clone this repository
2. Download **LocalAssets** folder from googledrive
3. Put it in the **Assets** folder

Note: Make sure the folder structure is correct, should not be LocalAssets/LocalAssets/(music, models, textures, etc.), it should be LocalAssets/(music, models, textures, etc.)

--- 

## File Structure

Keep file structure the same without changing much. You can add another folder for more organization if needed, but **do not** make a folder like "Grants-Scripts" just put it in a normal folder please.

Attach a Header to the top of every script so we can see who edited what, here is a sample for use. This isn't to see who messed up, it's to see who we need to ask question to if we edit the same file later.
```C#
/*=============================================================================
Script Name:    <ScriptName>
Last Edited:    <YYYY-MM-DD>
Contributors:   <Name1>, <Name2>, ...
Description:    <Brief description of what this script does>
Notes:          <Any notes neccesary, you can remove this if not needed>
=============================================================================*/
```

Add a simple header on top of a function just with your initials and a brief description of the function
```C#
/* GH - <Brief description of what this function does> */
```
